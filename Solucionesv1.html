Facilidad de Distraccion

Ricardo Palma & Christian Moreno

Dia a dia, uno se encuentra con el inconveniente donde esta haciendo algo importante o tiene algun pendiente pero este se le olvida. Ya sea por que esta muy ocupado, es a largo plazo el pendiente o si es muy distraido.

El objetivo de este producto es dar a mostrar su uso para aquellas personas que se distraen con facilidad.
